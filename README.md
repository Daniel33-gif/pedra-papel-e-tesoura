# pedra-papel-e-tesoura
Este é um projeto simples de Pedra, Papel e Tesoura desenvolvido em Python. O jogo permite que o usuário jogue contra o computador diretamente pelo terminal. O programa gera uma escolha aleatória para o computador, compara com a escolha do jogador e mostra o resultado da rodada. O objetivo do projeto é praticar lógica de programação.
